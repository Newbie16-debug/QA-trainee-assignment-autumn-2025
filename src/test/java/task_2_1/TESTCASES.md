## TC-01 — Успешное создание задачи

### Предусловия:
1. Открыть страницу https://avito-tech-internship-psi.vercel.app.

### Шаги:
1. Нажать кнопку «Создать задачу».
2. Ввести название.
3. Ввести описание.
4. Выбрать проект.
5. Выбрать приоритет.
6. Выбрать исполнителя.
7. Нажать «Создать».

### Ожидаемый результат:
- Модальное окно закрывается. 
- Новая задача появляется в списке.


## TC-02 — Кнопка имеет состояние disabled, если обязательные поля не заполнены

### Шаги:

1. Открыть модалку создания задачи.
2. Заполнить только описание.
3. Не заполнять поле «Название».

### Ожидаемый результат:

- Кнопка «Создать» disabled.
- Поле «Название» подсвечено как обязательное.
