### Bug Report-01 — Reload страницы приводит к ошибке 404


Заголовок: Обновление страницы приводит к ошибке 404
Описание:
При нахождении на любых страницах , после нажатия F5 / Ctrl+R сервер возвращает 404 Not Found.

### Шаги для воспроизвдения:

1. Перейти на страницу https://avito-tech-internship-psi.vercel.app

Нажать F5

Получить ошибку 404

Ожидаемый результат:
Страница должна загрузиться корректно.

Фактический результат:
Появляется ошибка 404 — серверная маршрутизация не обрабатывает запрос.

Приоритет: High
Серьезность: Critical
— пользователь теряет страницу; нормальная работа невозможна.



### Bug Report-02 — Невозможно уникально идентифицировать задачу


Заголовок: Отсутствует уникальный идентификатор задачи в UI и DOM
Описание:
Задача не имеет уникального ID или его отображения.
Из-за этого невозможно:

однозначно выбрать элемент в DOM,

отличить задачи с одинаковыми названиями,

автоматически тестировать выбор задачи, селекторы нестабильны,

передавать прямую ссылку на конкретную задачу.

Шаги воспроизведения:

Создать две задачи с одинаковым названием

Посмотреть список задач

Обратить внимание, что отличить их невозможно

Ожидаемый результат:
У задачи должны быть:

уникальный идентификатор (id)

отображение ID в UI (например #123)

либо data-test-id в DOM

Фактический результат:
ID скрыт или отсутствует, задачи невозможно различить.

Приоритет: High
Серьезность: Major
— функционал в целом работает, но тяжело тестировать и появляются логические ошибки.






### Bug Report-03 — Скроллбар появляется на мониторе с разрешением 2560 × 1440 и интерфейс сдвигается


Заголовок: Скроллбар появляется на экранах 1440p+, приводя к горизонтальному смещению интерфейса
Описание:
При разрешении 2560×1440 (2К), когда список задач можно листать, появляется вертикальный скроллбар. Из-за этого часть интерфейса смещается вправо.

### Шаги для воспроизвдения:

1. Открыть приложение на мониторе 2560×1440

2. Открыть страницу «Все задачи»

3. Обратить внимание на появление скроллбара

### Ожидаемый результат:
Страница корректно адаптируется под большие экраны, скроллбар не появляется без необходимости.

### Фактический результат:
Появляется вертикальный скроллбар, смещение интерфейса.

### Окружение:

Windows 11

Chrome 121

Монитор 2560×1440

### Приоритет: Low
### Серьезность: Minor
