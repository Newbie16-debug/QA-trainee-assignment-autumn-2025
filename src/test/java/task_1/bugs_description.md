## 1) Ложный пустой статус ("Ничего не нашлось") внутри списка
- **Описание:** между блоками вакансий выводится сообщение «Ничего не нашлось», хотя вакансии присутствуют.
- **Ожидаемый результат:** сообщение "Ничего не нашлось" скрыто, показывается список вакансий под выбранными фильтрами.
- **Фактический результат:** между блоками вакансий ошибочно выводится сообщение «Ничего не нашлось».
- **Приоритет:** High

## 2) Дублирующийся фильтр "Data Science"
- **Описание:** в строке фильтров тег «Data Science» виден два раза.
- **Ожидаемый результат:** каждый выбранный фильтр отображается один раз.
- **Фактический результат:** фильтр «Data Science» виден два раза.
- **Приоритет:** Medium

## 3) Некорректная подсветка выбранного фильтра "Поддержка пользователей"
- **Описание:** выделенный фильтр "Поддержка пользователей" показан в чёрном стиле. Все остальные выбранные фильтры белого цвета.
- **Ожидаемый результат:** единый цвет отображения выделенных фильтров.
- **Фактический результат:** выделенный фильтр "Поддержка пользователей" отображается в чёрном стиле.
- **Приоритет:** Low

## 4) Картинка в правом верхнем углу не загружена
- **Описание:** справа от поска видно картинку с пустым изображением.
- **Ожидаемый результат:** картинка корректно отображается.
- **Фактический результат:** справа от поска видно картинку с пустым изображением.
- **Приоритет:** Low

## 5) Города в вакансиях не соответствуют выбранному городу-фильтру
- **Описание:** в списке вакансий города не соответствуют выбранному фильтру. Выбран город Санкт-Петербург -> отображаются дополнительно: Москва, Самара и вакансия с пустым городом.
- **Ожидаемый результат:** вакансии по фильтру содержат город, выбранный пользователем в запросе.
- **Фактический результат:** отображаются вакансии с городами, не выбранными пользователем.
- **Приоритет:** Medium
